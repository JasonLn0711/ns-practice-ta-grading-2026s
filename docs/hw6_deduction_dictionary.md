# HW6 Deduction Dictionary Index

HW6 uses separate deduction dictionaries for the two independent official scores.

- `HW6(code)`: see `docs/hw6_code_deduction_dictionary.md`
- `HW6(圖)`: see `docs/hw6_figure_deduction_dictionary.md`

Every non-full category score must have a written deduction reason and a tag from the relevant dictionary. Possible plagiarism or suspicious duplication is recorded as manual review only; this workspace does not decide academic-integrity guilt automatically.
